How to Use This Slideshow

  1. Copy the code - Copy the entire HTML, CSS, and JavaScript below the slideshow container.
  2. Paste into your website - Insert the code where you want the slideshow to appear on your school website.
  3. Customize images - Replace the image URLs in the background-image property of each slide.
  4. Update text - Change the titles and text in each slide's overlay to match your school's messaging.
  5. Adjust timing - Modify the slideInterval variable in the JavaScript to change how long each slide displays (default: 6 seconds).

Note: This slideshow has no visible controls or arrows as requested. It automatically cycles through images with smooth scale transitions. Images will gently zoom in during display for a subtle, engaging effect.
